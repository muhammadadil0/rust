# Rust Learning Resources

This repository contains Rust learning materials and exercises focused on fundamental concepts.

## Contents

- **variable_to_borrowing.zip** - A comprehensive guide covering Rust fundamentals from variables to ownership and borrowing concepts.

## Topics Covered

This resource explores:
- Variables and mutability in Rust
- Ownership and the borrow checker
- References and borrowing
- String types and memory management
- Pattern matching and error handling

## Getting Started

1. Extract the `variable_to_borrowing.zip` file to access the learning materials
2. Review the concepts in order for a structured learning experience
3. Practice the exercises to reinforce understanding

## Prerequisites

- Basic programming knowledge
- Rust installed on your system (download from [rustup.rs](https://rustup.rs/))
